# 台灣彩券全自動分析系統

一個完全自動化的台灣彩券開獎號碼分析系統，每日自動更新最新開獎資料。

## ✨ 功能特色

- ✅ **全自動更新**：每日自動下載最新開獎資料
- ✅ **無需維護**：設定一次，永久自動運行
- ✅ **多遊戲支援**：大樂透、威力彩、今彩539、雙贏彩等
- ✅ **歷史紀錄**：完整保存所有開獎歷史
- ✅ **熱門分析**：自動計算熱門號碼
- ✅ **完全免費**：使用 GitHub 免費服務

## 🚀 快速開始

1. **訪問網站**：系統部署完成後可直接訪問
2. **選擇遊戲**：從下拉選單選擇要分析的彩券類型
3. **查看分析**：系統會自動顯示最新開獎和統計資料

## 🔄 自動化流程

系統每天自動執行以下流程：
1. 22:30（台灣時間）自動啟動
2. 訪問台灣彩券官方網站
3. 下載最新開獎資料
4. 解析並轉換為 JSON 格式
5. 更新網站資料
6. 發送更新通知（可選）

## 📊 資料來源

資料來自台灣彩券官方網站：
- https://www.taiwanlottery.com/lotto/history/result_download

- **自動化**：GitHub Actions + Python
- **部署**：GitHub Pages
- **資料儲存**：JSON 格式
